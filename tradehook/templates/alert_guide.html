Alert Guide

ALL ORDER TYPES WILL REQUIRE YOUR TRADEHOOK API KEY

Alpaca

Simple Orders

Simple orders require a ticker, quantity, side, and time in force. These can be static:

alert('{"ticker": "SPY", "qty": 1, "side": "buy", "time_in_force": "gtc" ,"broker_name": "Alpaca - Paper", "order_type": "simple_order","tradehook_api_key":"YOUR-API-KEY"}',freq=alert.freq_once_per_bar_close)

Or parameterized based on your script variables or built-in pinescript variables. Numeric values MUST be converted to strings:

alert('{"ticker": "' + syminfo.ticker + '", "qty": ' + str.tostring(quantity) + ', "side": "buy/sell", "time_in_force": "gtc/other values" ,"broker_name": "Alpaca - Paper", "order_type": "simple_order","tradehook_api_key":"YOUR-API-KEY"}',freq=alert.freq_once_per_bar_close)

Limit Orders

Limit orders require everything that a simple order requires in addition to a limit price (limit_price):

('{"ticker": "' + syminfo.ticker + '", "qty": ' + str.tostring(quantity) + ', "limit_price": ' + str.tostring(your_limit_price) + ',"side": "buy/sell", "time_in_force": "gtc/other values" ,"broker_name": "Alpaca - Paper", "order_type": "limit_order","tradehook_api_key":"YOUR-API-KEY"}',freq=alert.freq_once_per_bar_close)
where your_limit_price is a variable in your pinescript code.

Stop Loss Orders

Stop Loss orders require everything that a simple order requires in addition to a stop loss:

alert('{"ticker": "' + syminfo.ticker + '", "qty": ' + str.tostring(quantity) + ', "stop_loss": ' + str.tostring(your_stop_loss_price) + ',"side": "buy/sell", "time_in_force": "gtc/other values" ,"broker_name": "Alpaca - Paper", "order_type": "stop_loss_order","tradehook_api_key":"YOUR-API-KEY"}',freq=alert.freq_once_per_bar_close)

Bracket Orders

Bracket orders combine stop loss orders with a take profit. They require the same information as simple orders with the addition of: bracket stop loss, bracket limit price, and bracket take profit:

alert('{"ticker": "' + syminfo.ticker + '", "qty": ' + str.tostring(quantity) + ', "bracket_stop_loss": ' + str.tostring(your_bracket_stop_loss) + ', "bracket_limit_price": ' + str.tostring(your_bracket_limit_price) + ',"bracket_take_profit": ' + str.tostring(your_bracket_take_profit) + ',"side": "buy/sell", "time_in_force": "gtc/other values" ,"broker_name": "Alpaca - Paper", "order_type": "bracket_order","tradehook_api_key":"YOUR-API-KEY"}',freq=alert.freq_once_per_bar_close)

Trailing Stop Orders

Trailing Stop orders require the same information as a simple order in addition to a trail price:

alert('{"ticker": "' + syminfo.ticker + '", "qty": ' + str.tostring(quantity) + ', "trail_price": ' + str.tostring(your_trail_price) + ',"side": "buy/sell", "time_in_force": "gtc/other values" ,"broker_name": "Alpaca - Paper", "order_type": "trailing_stop_order","tradehook_api_key":"YOUR-API-KEY"}',freq=alert.freq_once_per_bar_close)

Stop Price Orders

Stop Price Orders require the same information as a simple order in addition to a stop price and a limit price:

alert('{"ticker": "' + syminfo.ticker + '", "qty": ' + str.tostring(quantity) + ',"stop_price": ' + str.tostring(your_stop_price) + ' ,"limit_price": ' + str.tostring(your_limit_price) + ',"side": "buy/sell", "time_in_force": "gtc/other values" ,"broker_name": "Alpaca - Paper", "order_type": "stop_price_order","tradehook_api_key":"YOUR-API-KEY"}',freq=alert.freq_once_per_bar_close)

Sell All

Sell All simply sells all assets of a specific ticker:

alert('{"ticker": "' + syminfo.ticker + '", "side": "sell", "time_in_force": "gtc" ,"broker_name": "Alpaca - Paper", "order_type": "sell_all", "tradehook_api_key":"YOUR-API-KEY"}',freq=alert.freq_once_per_bar_close)

Above works for stocks, there are some exceptions for crypto
Due to fees on crypto purchases, selling quantities of crypto is more involved.
For a simple order to sell crypto use:
"order_type": "crypto_simple_sell"
for limit sells"
"order_type": "crypto_limit_sell"
for stop price sells:
"order_type": "crypto_stop_price_sell"
for selling all of a specific crypto:
"order_type": "crypto_sell_all"
