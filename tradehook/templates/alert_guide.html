    Webhook Format

  • alert('{"ticker": "' + syminfo.ticker + '", "qty": "1", "side": "buy", "time_in_force": "gtc", "broker_name": "Alpaca", "order_type": "bracket_order", "bracket_stop_loss": ' + str.tostring(bracket_stop_loss) + ',"bracket_limit_price": ' + str.tostring(bracket_limit_price) + ',"bracket_take_profit": ' + str.tostring(bracket_take_profit) + ',"tradehook_api_key":"Enter Your TradeHook API Key"}',freq=alert.freq_once_per_bar_close)
  • Json message returned: {"ticker": "LTCUSD", "qty": "1", "side": "buy", "time_in_force": "gtc", "broker_name": "Alpaca", "order_type": "bracket_order", "bracket_stop_loss": 60.8355,"bracket_limit_price": 60.221,"bracket_take_profit": 62.679, "tradehook_api_key":"Enter Your TradeHook API Key"}

    Order Types and Parameters for Stock Orders

  • simple_order : 'ticker', 'qty', 'side', 'time_in_force'
  • limit_order : 'ticker', 'qty', 'side', 'time_in_force', 'limit_price'
  • stop_loss_order : 'ticker', 'qty', 'side', 'time_in_force', 'stop_loss'
  • bracket_order : 'ticker', 'qty', 'side', 'time_in_force', 'bracket_stop_loss','bracket_limit_price','bracket_take_profit'
  • trailing_stop_order : 'ticker', 'qty', 'side', 'time_in_force', 'trail_price'
  • stop_price_order : 'ticker', 'qty', 'side', 'time_in_force', 'stop_price', 'limit_price'

Crypto buys through Alpaca are only compatible with simple, limit, and stop_price orders. You must also use different order types to sell Crypto due to fees.

    Order Types and Parameters for Crypto Sell Orders

  • crypto_simple_sell : 'ticker', 'qty', 'time_in_force'
  • crypto_limit_sell : 'ticker', 'qty', 'time_in_force', 'limit_price'
  • crypto_stop_price_sell : 'ticker', 'qty', 'time_in_force','limit_price','stop_price'
