Welcome to TradeHook!

This site functions as a middleman between any webhook sender and compatible Brokerages. Upon receiving a webhook this system will automatically process it and send it to the connected brokerage. You will need to generate an api key for this site and include it in the webhook message. You will also have to add brokerages to your account and insert your brokerage api key(s). This site also includes an event log that contains the raw webhook message as well as the brokerage response. This site is very much a work in progress so I will be adding a feedback tab soon. I will be adding brokerages and possibly other features periodically. Also, currently this site only accepts paper trading endpoints and api keys since I am inexperienced with security.
