Setup Guide

  1. First you will need to sign up for or log into your broker account.
  2. Then, unless you already have them, you will need to generate API keys for this broker.
  3. Next, you will go to the "My API Keys page." Here you will generate an API key for Tradehook.
  4. You will then select "Add Broker" and select your broker and enter the associated API Keys.
  5. Finally, open TradingView (or another charting platform/webhook provider).
  6. Create an alert based on your specified conditions.
  7. Under the notifications tab of your alert, you will see the "Webhook Url" option.
  8. Enable that option and enter this URL: https://tradehook-prod-fbb7997de66a.herokuapp.com/tradehook/webhook/
  9. You should test that the alert is working and that the format is JSON compatible.
  10. You can create and delete Tradehook and broker API keys at any time.
  11. You can view your sent webhooks as well as broker responses at the event log page.
