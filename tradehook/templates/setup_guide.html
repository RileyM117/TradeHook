Broker Setup

First you will need to sign up for or log into your broker account.
Then, unless you already have them, you will need to generate API keys for this broker.

Next, You will navigate to the API Keys page on TradeHook. You will then select "Add Broker" and select your broker and enter the associated API Keys.

TradeHook API Key

Then, you will generate a TradeHook API Key.

TradingView Setup

After that, open TradingView (or another charting platform/webhook provider) and create an alert based on your specified conditions.

Finally, Under the notifications tab of your alert, you will see the "Webhook Url" option.
Enable that option and enter this URL: https://tradehook-prod-fbb7997de66a.herokuapp.com/tradehook/webhook/
